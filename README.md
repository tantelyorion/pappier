# Pappier - Éditeur de Code Simple

Pappier est un éditeur de code minimaliste écrit en C pur, avec un thème sombre inspiré de Vezo Browser.

## 🎨 Thème

Pappier utilise la palette de couleurs Vezo :
- **Principal** : #2F80FF
- **Accent** : #00D4FF
- **Fond** : #080A0F
- **Surface** : #161B22
- **Texte** : #F5F7FA

## ✨ Fonctionnalités

- [x] Édition de texte simple
- [x] Coloration syntaxique (C, Python, JavaScript)
- [x] Navigation au clavier
- [x] Mode Normal et Mode Insertion
- [x] Commandes (sauvegarde, ouverture)
- [x] Thème personnalisé

## 🚀 Installation

```bash
# Compiler
make

# Installer (optionnel)
